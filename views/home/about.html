MongoRestController

To enable rapid REST API prototyping you can use microscope mongo-rest-controller NPM package. 

npm i mongo-rest-controller --save
	
import MongoRestController from 'mongo-rest-controller';
import Article from '../models/Article';

export default class ArticleController extends MongoRestController {
    get baseUrl(){
        return '/api/articles'
    }

    get model(){
        return Article;
    }
}
